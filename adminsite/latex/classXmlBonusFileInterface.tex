\subsection{Detailed Description}
A class used to manage the content of a Bonus\-File

A Bonus\-File contains several Choices. Each choice has a name and a description. A choice contains a list of Attribute\-Modifier that is a pair of Attribute/Modifier.

\item[See also:]\doxyref{Xml\-Bonus\-File\-List\-Interface}{p.}{classXmlBonusFileListInterface} \end{Desc}
