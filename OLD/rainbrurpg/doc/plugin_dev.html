Here, you can find help on plugins development. I will try to show you the class you can use, and how you can use it.

Prerequisites

To make a plugin for RainbruRPG, you must know C++ programmation rules. Making a dynamic library (.dll under win32, .so under *nix) is also needed.

Options integration

RainbruRPG lets you add your own options to the launcher option editor. We will add a boolean option in a custom optionButton. 
// First, we get the 'plugins' OptionPanel
OptionPanel* pluginsPanel=OptionManager::getSingleton().getPluginsPanel();

// We create our custom button (our plugin is called Foo and we choose an icon
OptionButton* btn1=new OptionButton("Foo", "../data/fox/dawntime.ico");

// We create the boolean option attribute
OptionBoolAttribute *boo=new OptionBoolAttribute("FooBool");
boo->setToolTip("A tooltip that helps the user :)");

btn1->add(boo);
