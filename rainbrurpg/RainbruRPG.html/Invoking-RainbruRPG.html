6 Invoking RainbruRPG

The format for running the RainbruRPG program is: 

     RainbruRPG option ...

RainbruRPG supports the following options:

--help
-?
Print an informative help message describing the options and then exit successfully.
--usage
Briefly list the options.
--version
-V
Print the version number of RainbruRPG on the standard output and then exit successfully.
