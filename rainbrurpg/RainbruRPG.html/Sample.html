5 Sample Output

Here are some realistic examples of running RainbruRPG.

This is the output of the command `RainbruRPG': 

     hello, world!

This is the output of the command `RainbruRPG --help': 

     Usage: RainbruRPG [OPTION...] [FILE...]
     RainbruRPG is a in-development Multiplayer Online Role-Playing Game
     
       -?, --help                 Give this help list
           --usage                Give a short usage message
       -V, --version              Print program version
     
     Report bugs to <rainbru@free.fr>.
