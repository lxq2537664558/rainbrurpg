Here is how are computed results :

Lines
The SLOC count is generated using David A Wheeler's 'SLOCCount'. This value must be computed on the src/ directory. Please note that before v0.0.5-199, this number counted also generated files as sources was not is src/ directory.
cc
represents the Console's Count command result. Not computed since the console is not accessible.
ut
represents the number of unit test (made with CppUnit and Boost.Test). To know how many Boost.Test based tests can be launched, please uses src/nb_unit_tests.pl.
vg
set to failed if valgrind reports possible memory leaks on unit tests for the given library/directory.
Sys
set to OK if the application can be run on the given OS. The compilation can be made on another system (cross-compilation).
PoMsg
the number of ready-to-be-translated strings. There are listed in the po/RainbruRPG.po. It was not computed before build 191.
Pass?
The overall result of the non-regression tests.
